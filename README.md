# trader-sentiment-analysis
Analysis of trader performance and behavior across Bitcoin Fear &amp; Greed market sentiment.
## Trader Behavior Analysis vs Market Sentiment

### Objective
Analyze how Bitcoin market sentiment (Fear & Greed Index) influences trader behavior and performance.

### Datasets
- Hyperliquid historical trader data
- Bitcoin Fear & Greed Index

### Approach
- Data cleaning and timestamp alignment
- Merging trader data with daily sentiment
- Analysis of profitability, win rate, trade size, and risk
- Visualization of sentiment-driven behavior

### Key Insights
- Greed phases show higher trading activity but increased volatility
- Win rates drop during Extreme Greed due to overconfidence
- Fear phases exhibit conservative trading with more stable outcomes
- Loss magnitude increases during extreme sentiment regimes

### Tools Used
Python, Pandas, Matplotlib, Seaborn
